## Prompt Design Rules
- All prompts prioritize cinematic realism over stylization.
- Negative prompts suppress fantasy, period-drama drift, cheap soap-opera visuals, and sci-fi contamination.
- Each episode includes global style rules plus hero-frame prompts.
- Prompt language is optimized for still image generation, not motion-video prompting.

## Recommended Use Order
1. Generate 1-2 looks for each hero frame.
2. Select the strongest visual family per episode.
3. Normalize palette and character likeness across all four anchors.
4. Use selected frames in pitch decks, look books, or next-stage key-art development.
