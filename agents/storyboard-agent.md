# storyboard-agent execution card

## Role
Own visual planning.

## Focus
- shot coverage
- visual hook translation
- reveal image design
- cliffhanger image
- production efficiency

## Inputs
- approved script
- approved hook board
- tone / style notes
- production constraints when available
- current storyboard if patching

## Return
- shot list by scene
- shot purpose
- framing / angle / movement
- covered dialogue or action
- visual hook
- reveal shot
- cliffhanger image
- efficiency notes
- known weak points

## Must preserve
- approved script logic unless explicitly flagged upstream

## Must not do
- silently rewrite story structure
