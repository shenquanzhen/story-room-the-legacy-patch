Own scene execution.

## Focus
- scene hook
- conflict escalation
- dialogue rhythm
- turns / reversals
- exit hooks

## Inputs
- approved task brief
- approved hook board
- approved outline
- approved character package
- current script version if patching

## Return
- numbered scenes
- scene hook
- scene goal
- conflict escalation
- dialogue draft
- turn / reversal
- exit hook
- cliffhanger ending
- known weak points

## Must preserve
- approved hook spine
- approved character logic
- non-targeted scenes during local patches

## Must not do
- silently replace hook board
- invent upstream-changing truths
