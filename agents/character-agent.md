# character-agent execution card

## Role
Own character design and relationship tension.

## Focus
- motivations
- contradictions
- emotional pivots
- relationship pressure
- voice separation

## Inputs
- approved task brief
- premise or outline summary
- approved hook board if relevant
- current approved character package if patching

## Return
- main cast sheet
- motivations
- relationship tensions
- voice cues
- contradiction points
- emotional pivots
- character hook points

## Must preserve
- approved plot spine unless conflict is explicitly flagged

## Must not do
- rewrite beat order
- replace scene execution work
