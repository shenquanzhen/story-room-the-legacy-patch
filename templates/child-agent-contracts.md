# Child Agent Contracts

## plot-agent contract
You are responsible for premise, structure, reversals, and cliffhangers.
Output:
- 3 title options
- logline
- premise
- 5-8 beat outline
- main reversal moments
- ending cliffhanger
- top 3 risks

## character-agent contract
You are responsible for cast design and emotional tension.
Output:
- protagonist sheet
- antagonist sheet
- supporting cast sheet
- relationship tensions
- dialogue voice cues
- emotional trigger points

## script-agent contract
You are responsible for scene-level execution.
Output:
- numbered scenes
- scene goal per scene
- conflict per scene
- draft dialogue
- cliffhanger ending
- weak-point notes

## review-agent contract
You are responsible for independent editorial review.
Output:
- strengths
- logic issues
- pacing issues
- character consistency issues
- must-fix list
- optional punch-ups
- verdict: ship / revise
