# 《完美继承人》场景—角色—集数三维索引表（全季）

> 用途：制片统筹、执行统筹、通告预排、演员档期协调、预算与置景复用判断
> 口径说明：本表以现有《制片拆景版》《场景清单版》为基础，按**制片可执行场景资产**重新归并，不按逐场文学命名逐条罗列，而按“可拍摄、可复用、可通告”的实际场景单元整理。
> 判断规则：
> - **高频复用**：涉及3集及以上，或同一场景体系内跨季反复使用。
> - **高能场**：该场景内承载重要反转、公开审判、关系破裂/修复、系统失控、终局抉择等核心戏。
> - **核心角色**：该场景的主驱动角色 / 高集中出勤角色。
> - **次要角色**：阶段性进入该场景、但非该景常驻驱动角色。

## 二、按角色反查：角色—场景索引

> 用途：快速判断某角色集中出现在哪些场景，便于演员档期、通告打包与陪戏安排。

### 1. 顾言成
- **高频主场景**：顾言成房间 / 书桌区 / 房门外、顾家客厅、顾家餐厅、家中书房 / 系统书房、集团会议室 / 家族会议厅、后台通道 / 直播后通道、系统后台 / 主控台 / 终端空间、安全隔离室
- **重点功能场景**：放学天台、旧游乐场、直播主舞台、顾言成与陆竞川对峙空间、隔离区外
- **统筹建议**：可拆为“顾家主宅组”“商务权力组”“直播/后台组”“终局系统组”四大拍摄包

### 2. 沈蔓宁
- **高频主场景**：顾家客厅、顾家餐厅、老爷子书房、夫妻卧室、车内、地下车库、后台通道 / 直播后通道、安全隔离室
- **重点功能场景**：母亲办公室 / 深夜资料空间、顾知微公寓、观察中心、记者见面会 / 公开声明现场、旧游乐场
- **统筹建议**：沈蔓宁重场高度集中在“真相线—认罪线—唤醒线—终局线”，建议按情绪阶段打包

### 3. 顾承泽
- **高频主场景**：顾家客厅、顾家餐厅、老爷子书房、家中书房 / 系统书房、夫妻卧室、地下车库、集团会议室 / 家族会议厅、后台通道 / 直播后通道、安全隔离室
- **重点功能场景**：顾承泽办公室、车内、空置别墅、隔离区外、记者见面会 / 公开声明现场
- **统筹建议**：建议按“父子战争组”“家族权力组”“认罪与终局组”集中拍摄

### 4. 顾启山
- **高频主场景**：顾家餐厅 / 家宴长桌、老爷子书房、顾家客厅、集团会议室 / 家族会议厅、顾家席位 / 直播席位区
- **统筹建议**：戏份不算最多，但场场关键，适合集中锁定 2—3 个拍摄周期完成

### 5. 顾承钧
- **高频主场景**：顾家餐厅 / 家宴长桌、老爷子书房、顾家客厅、集团会议室 / 家族会议厅、顾家席位 / 直播席位区
- **统筹建议**：与顾启山、梁佩珊绑定统筹价值高，适合家族群戏打包通告

### 6. 顾星遥
- **高频主场景**：顾家餐厅 / 家宴长桌、老爷子书房、顾家客厅、顾星遥介入空间 / 夜谈点位

### 8. 陆竞川
- **高频主场景**：实验室 / 会客间、家中书房关联场、直播彩排后台、直播后台控制区、系统后台 / 主控台 / 终端空间、顾言成与陆竞川对峙空间

### 9. 韩若岚

### 12. 许临

## 四、制片执行结论

### 1. 全季最高复用场景资产
按制片价值排序，建议优先锁定：
1. **顾家主宅整套**（客厅、餐厅、书房、卧室、顾言成房间、走廊、旧物区）
2. **集团会议室 / 办公室体系**
3. **系统后台 / 隔离室 / 后台通道体系**
4. **直播主会场 + 顾家席位 + 后台控制区**
5. **车内 / 地下车库夜戏体系**

### 2. 最适合并组拍摄的场景包
- **顾家主宅包**：1、2、3、4、5、6、7、8 号场景
- **校园包**：13、14、15、16、17、18 号场景
- **商务权力包**：19、20、21 号场景
- **曝光公域包**：23、25、26、27、28、32、33、34 号场景
- **系统终局包**：31、35、36、37 号场景
- **夜戏交通包**：9、10 号场景

### 3. 最需要单独保护的高能场景体系
- **顾家客厅 / 会客厅**：家庭关系与家族神话双重崩裂
- **集团会议室 / 家族会议厅**：继承考核与家规反噬
- **后台通道 / 直播后通道**：21—22集连续母子唤醒主阵地
- **大型直播现场主舞台**：21集全季最大公共爆点
