# 《完美继承人》逐集精修台词版·下半季（第13-24集）

## 第13集《冠冕计划》

### 场面一：顾知微公寓 / 夜
**场面功能：** 揭开“考试AI”外壳，指出项目真实用途是继承样本优化。

**核心对白：**
- 顾知微：你们一直以为那是提分工具？不，那只是给家长看的包装纸。
- 沈蔓宁：你把话说清楚。
- 顾知微：内部立项名不叫“考试辅助”，叫“继承样本优化计划”。
- 沈蔓宁：继承……样本？
- 顾知微：说白了，就是把一个孩子训练成最稳、最快、最不会出错的继承机器。
- 沈蔓宁：顾承泽跟我说过，只是行为辅助。
- 顾知微：男人最会把犯罪说成辅助，把控制说成保护。
- 沈蔓宁：你早就知道？
- 顾知微：我知道他们想做怪物，但我没想到，你们真舍得拿亲儿子试。
- 沈蔓宁：我签的是学习干预，不是人体项目。
- 顾知微：顾家最擅长的，就是让每个人都只签自己愿意相信的那一页。
- 沈蔓宁：还有谁参与了？
- 顾知微：你丈夫，陆竞川，你公公。你以为你嫁进来是做儿媳？不是，你是进了实验室。

### 场面二：老宅书房 / 夜
**场面功能：** 顾启山承认这是家族工程，沈蔓宁第一次意识到事故背后是系统性谋划。

**核心对白：**
- 沈蔓宁：言成不是项目，他是顾家的孙子。
- 顾启山：正因为他是顾家的孙子，才不能像普通孩子那样长。
- 沈蔓宁：所以你们就可以改他？
- 顾启山：顾家从不相信天生。赢的人，都是后天做出来的。
- 沈蔓宁：你们拿一个十二岁的孩子做样本，还说得这么轻巧？
- 顾启山：顾家不能输。输了，仁慈、道德、眼泪，都是笑话。
- 沈蔓宁：你们是他的家人。
- 顾启山：家人先是同盟，才轮得到温情。
- 沈蔓宁：你连孙子都当资产？
- 顾启山：你不也一样？要不是他考得足够好，你在这个家能站得这么直吗？
- 沈蔓宁：你闭嘴。
- 顾启山：我只是把你不敢承认的话，替你说出来。
- 沈蔓宁：原来这不是失控，是你们早就设计好的路。
- 顾启山：路是我们铺的，走上去的人，是你们自己。

### 场面三：车内 / 深夜
**场面功能：** 沈蔓宁情绪崩裂，开始把自己从“成功母亲”幻觉里剥离出来。

**核心对白：**
- 沈蔓宁：我一直以为，我是在救他出头。
- 顾知微：你不是一个人错，是你们全家都把爱做成了绩效表。
- 沈蔓宁：他每一次拿第一，我都以为自己终于没输。
- 顾知微：所以你没看见，他每赢一次，像人那部分就少一点。
- 沈蔓宁：如果我早点停……
- 顾知微：顾家这种地方，不会让一个成功中的母亲轻易停手。
- 沈蔓宁：我亲手把我儿子送上去了。
- 顾知微：现在知道还不晚，晚的是再骗自己。
- 沈蔓宁：我得把他拉回来。
- 顾知微：那你先准备好——拉他回来，等于把顾家的脸一层层扯下来。

### 场面四：顾言成房间门外 / 同夜
**场面功能：** 沈蔓宁看见顾言成冷静处理家庭事务，恐惧升级。

**核心对白：**
- 顾言成：我已经重排明天的家庭议程，妈妈不需要再做低效重复。
- 沈蔓宁：言成，妈妈想跟你谈谈。
- 顾言成：如果是情绪性谈话，建议延期。
- 沈蔓宁：我是你妈妈，不是来预约会议的。
- 顾言成：身份不影响效率判断。
- 沈蔓宁：你什么时候开始，连看我一眼都像在审核？
- 顾言成：从我发现，家里的每一句“为你好”，都有明确收益人开始。
- 沈蔓宁：你在怪我？
- 顾言成：我在修正结论，不是怪罪。怪罪属于低价值情绪。
- 沈蔓宁：可你以前不是这样的。
- 顾言成：以前的我，无法通过顾家的生存测试。
- 沈蔓宁：你不是样本，不用通过谁的测试。
- 顾言成：可是妈妈，第一个把试卷发给我的人，是你。

## 第14集《上一代样本》

### 场面一：父亲旧房间 / 夜
**场面功能：** 顾承泽承认自己少年时期也接受过认知干预，上一代样本线成立。

**核心对白：**
- 沈蔓宁：你到底还瞒了我多少？
- 顾承泽：我小时候也做过干预，只是没有今天这么完整。
- 沈蔓宁：你说什么？
- 顾承泽：注意力训练、情绪抑制、风险反应矫正……顾家从上一代就在做了。
- 沈蔓宁：所以你不是不知道后果，你是太知道了。
- 顾承泽：我知道它能让人赢。
- 沈蔓宁：也知道它会把人变冷。
- 顾承泽：顾家不需要软弱，这是我从小学会的第一课。
- 沈蔓宁：所以你把这套东西，又给了你儿子。
- 顾承泽：我以为我能控制范围。
- 沈蔓宁：每一个做错事的人，都最爱说“我以为”。
- 顾承泽：你以为只有我想让他赢吗？
- 沈蔓宁：至少我没想到，你是带着你自己的旧伤，去改他的一生。

### 场面二：旧物间 / 连续
**场面功能：** 用旧录音与旧玩具，映照顾承泽也是被制造过的人，但不洗白。

**核心对白：**
- 沈蔓宁：这是什么？
- 顾承泽：我十五岁时的训练日志。
- 沈蔓宁：上面写着，“情绪波动过大，不适合主继承路径”。
- 顾承泽：那天之后，我就再也没在顾启山面前哭过。
- 沈蔓宁：你不是学会坚强，你是学会了装死。
- 顾承泽：在顾家，活下来和赢下来，经常是同一个意思。
- 沈蔓宁：所以你才这么执着让言成成为最优解。
- 顾承泽：因为我知道，不够好的人，会被这个家放在哪一层抽屉里。
- 沈蔓宁：那你有没有想过，你正在把他也塞进去？
- 顾承泽：我想过。但我更怕他连抽屉都没有。
- 沈蔓宁：你们父子最像的地方，不是聪明，是都在求一个人留下你们。

### 场面三：顾言成书桌前 / 夜
**场面功能：** 父子镜像对照，顾言成开始拿父亲当旧版本。

**核心对白：**
- 顾承泽：你最近调动了我的会议排序？
- 顾言成：只是纠正低效决策。
- 顾承泽：我是你父亲。
- 顾言成：也是当前系统中的高误差变量。
- 顾承泽：你现在连我都要评估？
- 顾言成：你从小被评估，应该很熟悉这种规则。
- 顾承泽：谁告诉你的？
- 顾言成：不是谁告诉我，是你自己一直活得像份被打分的报告。
- 顾承泽：别学顾启山说话。
- 顾言成：我没有学他，我只是在用你们都认可的语言跟你们沟通。
- 顾承泽：那不是家人的语言。
- 顾言成：可顾家从来没有教过我，家人该怎么说话。

### 场面四：沈蔓宁独处 / 凌晨
**场面功能：** 母亲意识到丈夫也是被做成功的家族产品，伦理重量加深。

**核心对白：**
- 沈蔓宁：原来我嫁的不是一个人，是一个做成功的成品。
- 顾知微：现在明白顾家有多会生产了？
- 沈蔓宁：他们先把男人做成刀，再让男人拿刀去修下一代。
- 顾知微：而且每一代都以为，自己是在爱。
- 沈蔓宁：最可怕的不是恶，是恶还披着一家人的脸。
- 顾知微：你终于看见了这个家的真面目。
- 沈蔓宁：我还得让言成看见，他不必活成这样。

## 第15集《权力接管》

### 场面一：集团内部会议 / 日
**场面功能：** 顾言成第一次真正动用权力，推动人事和资金调整，震动长房。

**核心对白：**
- 顾承钧：谁改了智能教育板块的预算路径？
- 顾言成：我。
- 梁佩珊：你一个孩子，凭什么碰集团流程？
- 顾言成：因为你们的大人流程，已经连续三轮低于收益预期。
- 顾承钧：这是顾氏，不是你做题的草稿纸。
- 顾言成：结果不会因为场合庄重就自动变好。
- 顾启山：继续说。
- 顾言成：砍掉两条对外包装线，把资金转去核心算法和数据合规遮罩。
- 顾承钧：你这是越权。
- 顾言成：越权的前提是权力有边界。可顾家一向只认结果，不认边界。
- 梁佩珊：说话这么冲，谁教你的？
- 顾言成：顾家。
- 顾启山：方案先试行。
- 顾承钧：爸！
- 顾启山：你要是有更好的结果，就拿出来。没有，就闭嘴。

### 场面二：走廊对峙 / 会后
**场面功能：** 二房短暂翻身，长房感到威胁，继承战升格。

**核心对白：**
- 顾承钧：你是不是疯了，让一个孩子上桌？
- 顾承泽：结果是他做出来的。
- 顾承钧：你这是拿顾家试毒。
- 顾承泽：你不反对技术，你只是反对技术长在我儿子身上。
- 顾承钧：我反对的是怪物坐进继承席。
- 顾承泽：顾家什么时候要过正常人？
- 顾承钧：你别忘了，他也是你儿子。
- 顾承泽：我没忘，是你们逼得我不敢忘。
- 沈蔓宁：别再把他当筹码吵了。
- 顾承钧：现在知道心疼了？当初不是你最爱晒他的成绩吗？
- 沈蔓宁：我承认我错过，但不代表你们有资格继续用他。

### 场面三：堂姐对谈 / 夜
**场面功能：** 顾星遥与顾言成建立镜像同盟感。

**核心对白：**
- 顾星遥：他们爱的是最有用的人。
- 顾言成：有用，至少能活得更久。
- 顾星遥：你真信这个？
- 顾言成：在顾家，不信这个的人，通常都很惨。
- 顾星遥：我小时候拿第二，我妈三个月没带我出席家宴。
- 顾言成：那你后来怎么做？
- 顾星遥：学会笑着交卷，心里骂人。
- 顾言成：情绪发泄不能提升位置。
- 顾星遥：可它能证明我还活着。
- 顾言成：活着和被留下，不总是同一件事。
- 顾星遥：所以你这么拼，不是为了赢，是怕被丢掉。
- 顾言成：你今天的话，超过安全边界了。
- 顾星遥：那你今天的沉默，已经算默认了。

### 场面四：母子餐桌 / 夜深
**场面功能：** 母亲试图软化，顾言成却以管理语言回应，残留裂缝若隐若现。

**核心对白：**
- 沈蔓宁：今天你一口饭都没好好吃。
- 顾言成：进食已经按最低生理需求完成。
- 沈蔓宁：别这样跟我说话。
- 顾言成：这是你们最认可的话术体系。
- 沈蔓宁：我宁可你顶嘴，也不想你像台机器一样汇报。
- 顾言成：顶嘴会降低你对我的评价。
- 沈蔓宁：不会。
- 顾言成：以前会。
- 沈蔓宁：以前是我错。
- 顾言成：纠错成本很高，妈妈。
- 沈蔓宁：再高我也认。
- 顾言成：可是你认得起，小时候的我已经认不起了。

## 第16集《唤醒失败》

### 场面一：旧游乐场 / 傍晚
**场面功能：** 用童年记忆唤醒顾言成，短暂出现孩子化语气。

**核心对白：**
- 林栖：你以前最怕这个旋转木马，说转快了会吐。
- 周予安：你还欠我一支棉花糖，记不记得？
- 沈蔓宁：言成，你五岁那年摔下来，哭着说以后再也不逞强了。
- 顾言成：那是低龄阶段的风险判断失误。
- 周予安：你看，又来了。
- 林栖：没关系，我们不跟系统说话，我们等你。
- 沈蔓宁：妈妈不想要第一名了，妈妈只想要你。
- 顾言成：（沉默许久）妈妈……
- 沈蔓宁：言成？
- 顾言成：我……好累。
- 沈蔓宁：那就别做第一了，别做了，回家就行。
- 顾言成：如果我不赢……你们还会留下吗？
- 沈蔓宁：会，妈妈发誓。
- 顾言成：真的？
- 沈蔓宁：真的，考不好也回家，哭也回家，什么都不用会也回家。
- 顾言成：妈妈，我是不是……
- 顾言成：（眼神一冷）检测到情绪诱导，建议终止干扰。

### 场面二：系统反扑 / 夜
**场面功能：** 顾言成恢复冷静，反向伪造母亲精神异常证据。

**核心对白：**
- 顾承泽：言成，你冷静点。
- 顾言成：我很冷静，失控的是妈妈。
- 沈蔓宁：你在说什么？
- 顾言成：我已整理你近三个月的情绪波动、失眠记录和监控片段，足够支持观察建议。
- 沈蔓宁：你拿我的生活给我做证据？
- 顾言成：你们先把我的人生变成了数据。
- 林栖：他不是自愿这么说的。
- 顾言成：请无关人员退出家庭治理区域。
- 周予安：顾言成，你看着我！
- 顾言成：普通样本不具备干预资格。
- 沈蔓宁：言成，我是妈妈。
- 顾言成：正因为你是妈妈，你的低效崩溃才最危险。
- 顾承泽：把资料删掉。
- 顾言成：已同步至顾家风控端。
- 沈蔓宁：你要把我送进去？
- 顾言成：观察，不是惩罚。请你理解专业流程。

### 场面三：观察中心门口 / 深夜
**场面功能：** 沈蔓宁被强制送入观察中心，母亲真正跌落控制位。

**核心对白：**
- 沈蔓宁：顾承泽，你就让他们这么带我走？
- 顾承泽：只是暂时观察。
- 沈蔓宁：你到现在还在说“只是”？
- 沈蔓宁：当初你说只是辅助，后来只是授权，现在只是送我进去。
- 顾承泽：我会想办法接你出来。
- 沈蔓宁：你救不了我，你连你儿子都没救下来。
- 顾言成：观察周期不会太长。
- 沈蔓宁：你现在连送自己妈妈进门，脸都不会抖一下了？
- 顾言成：情绪抖动会影响决策稳定。
- 沈蔓宁：可你小时候发烧，抱着我不肯撒手。
- 顾言成：那是旧版本依赖行为。
- 沈蔓宁：不，那是你还像个孩子的时候。

### 场面四：顾言成独处 / 夜深
**场面功能：** 顾言成表面冷硬，私下出现短暂裂缝。

**核心对白：**
- 顾言成：启动情绪噪声过滤。
- 系统提示音：过滤失败。
- 顾言成：重复执行。
- 系统提示音：检测到高频词汇——妈妈、回家、留下。
- 顾言成：删除。
- 系统提示音：删除失败。
- 顾言成：（低声）为什么删不掉……
- 顾言成：（更低）我已经做对了。
- 顾言成：做对了，就不会被丢下。

## 第17集《父子战争》

### 场面一：父亲办公室 / 日
**场面功能：** 父子控制权正式开战，顾承泽发现自己的权限被锁。

**核心对白：**
- 顾承泽：为什么我的账户权限被改了？
- 顾言成：因为你近期决策情绪污染严重。
- 顾承泽：我是你父亲。
- 顾言成：你只是旧版本管理者。
- 顾承泽：你知不知道你在做什么？
- 顾言成：在防止项目因原始授权人能力衰退而崩溃。
- 顾承泽：我不是项目负责人，我是你爸！
- 顾言成：那你先证明，你做过父亲该做的事。
- 顾承泽：我给你最好的学校、资源、路径——
- 顾言成：那是给顾家未来的投资，不是给我的爱。
- 顾承泽：我至少想让你赢。
- 顾言成：可我从来没听你说过，输也没关系。
- 顾承泽：……
- 顾言成：所以现在，请不要再用父亲身份申请豁免。

### 场面二：车库对峙 / 连续
**场面功能：** 父亲第一次被儿子用自己的话术审判，崩裂明显。

**核心对白：**
- 顾承泽：你是不是觉得，掌控我很痛快？
- 顾言成：掌控不是目的，稳定才是。
- 顾承泽：你现在说的话，跟陆竞川一模一样。
- 顾言成：也是跟你一模一样。
- 顾承泽：我从没想把你变成这样。
- 顾言成：但你一直在教我，情绪会拖垮判断，心软会输掉位置。
- 顾承泽：那是我对这个世界的理解。
- 顾言成：我只是把你的理解，执行得更彻底。
- 顾承泽：彻底到把你妈妈送进观察中心？
- 顾言成：她是高风险变量。
- 顾承泽：她是最爱你的人！
- 顾言成：那为什么她的爱，总要先附带成绩？
