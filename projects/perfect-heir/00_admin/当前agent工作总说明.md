##### 1. 场景清单版
用于制片前置统筹、执行导演、场务快速掌握：
- 重点场景
- 内外景
- 日夜
- 主要人物
- 核心动作
- 剧情功能
- 高能场标记

##### 2. 制片拆景版
用于资源整合与拍摄规划：
- 主场景
- 可复用场景
- 临时场景
- 群戏场
- 重点演员场
- 高能场
- 场景整合建议
- 并组拍摄建议
- 优先级保护

##### 3. 拍摄日程包·简版通告拆分表
用于按“资源包/拍摄包”重排拍摄逻辑：
- 拍摄包编号
- 拍摄主题
- 涉及集数与场景
- 核心演员
- 群戏 / 高能标记
- 适合并组的制片建议

##### 4. 场景—角色—集数三维索引表
用于快速反查：
- 某个场景跨哪些集
- 某个角色覆盖哪些场景
- 哪些场景高频复用
- 哪些场景高能且需单独保护

##### 5. 人物—集数—场景通告集中度表
用于判断：
- 演员通告集中度
- 哪些角色适合集中拍
- 哪些角色需要高能保护场单独安排

##### 6. 场景包并组拍摄建议表
用于制片压缩天数与资源复用：
- 哪些可合拍
- 哪些必须拆开保护
- 风险提示

##### 7. 高能场保护清单（执行版）
用于保护全季关键事件包：
- 母子创伤包
- 唤醒失败包
- 父子战争包
- 公开认罪包
- 直播审判包
- 终局选择包

并已给出：
- 拍摄风险
- 执行建议
- 哪些必须锁词 / 清场 / 控噪 / 预排练 / 留补拍口
