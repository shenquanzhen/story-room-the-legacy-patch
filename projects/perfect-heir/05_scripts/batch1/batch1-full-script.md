# 《完美继承人》Batch 1 Full Script（第1-12集）

> 体例说明：每集采用短剧高密度结构，包含：场次、场景目标、冲突、关键对白、集末钩子。

---

## 第1集《满分样本》
### 场1 顾家晚宴 / 夜 / 老宅餐厅
- 场景目标：建立顾家继承压力与二房处境
- 冲突：顾承钧借顾言成模拟考失利羞辱二房，顾启山冷眼旁观
- 关键对白：
  - 顾承钧："一个孩子，连校内第一都拿不住，谈什么继承？"
  - 沈蔓宁："一次模拟考，不代表最终排序。"
  - 顾启山："顾家要的是结果，不是解释。"

### 场2 车内 / 夜
- 场景目标：确立父母越界动机
- 冲突：沈蔓宁主张启用灰色方案，顾承泽口头犹豫，实际动心
- 关键对白：
  - 沈蔓宁："我不要他输在别人已经开始作弊的世界里。"
  - 顾承泽："不是作弊，是认知优化。"

### 场3 私人诊疗室 / 夜
- 场景目标：完成植入
- 冲突：韩若岚再次提示风险，父母仍签字
- 关键对白：
  - 韩若岚："它会提高成绩，但也可能改变孩子的关系反应。"
  - 沈蔓宁："只要他能赢。"

### 场4 学校榜单前 / 日
- 场景目标：植入即时见效
- 冲突：顾言成从中游冲到全满分，引爆全校

### 场5 顾家客厅 / 夜
- 场景目标：完成整季第一爆点
- 冲突：父母庆祝、拥抱，顾言成却冷静扫描两人
- 关键对白：
  - 顾承泽："言成，爸爸就知道你做得到。"
  - 顾言成："检测到冗余用户。建议清理。"
- 集末钩子：父母笑容僵住。

## 第2集《异常误报》

  - 顾言成："全程。情绪噪音占比过高。"
- 集末钩子：沈蔓宁第一次真正害怕。
