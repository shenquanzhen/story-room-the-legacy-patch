# story-room season review protocol

This file defines review logic above the single-episode layer.

Goal: detect season-scale weaknesses before they spread across many episodes.

---

## 1. Review layers

Season review should operate at three levels:
- season skeleton review
- batch checkpoint review
- late-season integrity review

---

## 2. Season skeleton review

Run before batch 1 drafting.

Check:
- season hook spine clarity
- protagonist arc logic
- antagonist escalation ladder
- midpoint strength
- final payoff promise
- episode role distribution
- reveal pacing skeleton

Output:
- what already works
- structural risks
- must-fix items by owner
- verdict: ship / revise

---

## 3. Batch checkpoint review

Run after each batch.

Check:
- continuity health
- escalation curve
- payoff reserve
- reveal discipline
- relationship progression
- cliffhanger diversity
- fatigue / repetition risk

Output:
- batch strengths
- continuity issues
- pacing drift issues
- season-level risks
- must-fix items by owner
- verdict: ship / revise

---

## 4. Late-season integrity review

Run before final episodes are locked.

Check:
- does final payoff still exceed earlier payoffs
- has the hidden power fantasy been overspent
- are betrayal / collapse turns earned
- are late reveals fresh enough
- does the finale repay the season promise

Output:
- final-stage strengths
- final-stage risks
- must-fix items by owner
- verdict: ship / revise

---

## 5. Severity guide

### Level 1
Single-episode weakness. Patch locally.

### Level 2
Batch-wide drift. Patch episode map or current batch.

### Level 3
Season-structural problem. Patch season spine first.

---

## 6. Routing rule

Send issues to the smallest responsible layer:
- season hook / season map problem -> season plot layer
- continuity logic problem -> continuity layer / episode map
- episode execution problem -> episode script layer
- storyboard translation problem -> storyboard layer

---

## 7. Autonomous gate rule

A batch should not open, or continue, if season review returns unresolved level 3 issues.
